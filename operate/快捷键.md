<h2><center>快捷键</center></h2>

### 1.idea

| Key mapping    | function          |
| -------------- | ----------------- |
| Alt  + ↑ / ↓   | 查看上/下一个方法 |
| ctrl + q       | 查看方法详情      |
| ctrl + f12     | 查看所有方法      |
| ctrl + n       | 查找类            |
| ctrl + f       | 当前类查找内容    |
| ctrl + alt + f | 全局查找内容      |

### 2.typera

| Key mapping      | function   |
| ---------------- | ---------- |
| ctrl + T         | 插入表格   |
| ctrl + 1-5       | 标题等级   |
| ctrl + shift + k | 生成代码块 |
