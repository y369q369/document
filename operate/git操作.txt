git操作：

1.切换到需要的目录下
2.下载项目： 	git clone 项目地址
3.查看分支：	git branch -a
4.切换分支：	git checkout -b develop --track remotes/origin/develop
				git checkout remotes/origin/wukun -b yangqing         
				
				
				
maven打包：		mvn install -Dmaven.test.skip=true
				mvn clean install
				mvn package -f pomWar.xml





修改文件
	git merge dev			与dev分支合并(当前分支为主)
	git status		  		查看修改文件
	git add 文件名    		添加修改的文件
	git commit -m '内容'	提交注释
	git push 				提交修改到github账户
